Hexo模板系统

Hexo的基本原理就是根据解析_post中的md格式文件，参照md文件中的布局类型根据模板解析。

Hexo使用node.js模板文件，例如ejs，swig，doT，Jade。Hexo中有post、page、photo等不同布局，选用的布局类型会在md文件中声明。

处理流程：Hexo首先解析md文件，然后根据layout.swig判断布局类型，再转发给其他的布局文件。
